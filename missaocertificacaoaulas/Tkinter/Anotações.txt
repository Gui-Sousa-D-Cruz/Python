
Importa a classe TK para uma variavel:
app = Tk() 
--------------------------------------------

Roda o loop principal do app. abrindo a janela:
app.mainloop() 
 
--------------------------------------------

Metodo para o titulo da janela:
app.title('#')

--------------------------------------------

Metodo para o tamanho da janela:
app.geometry('larg x alt')

--------------------------------------------

Metodo que configura a janela:
app.configure()

--------------------------------------------

Cria um widget do tipo label:
Label(app,text='')

--------------------------------------------

Metodo que posiciona o label criado:
.place(x=#,y=#,width=#,height=#) 

--------------------------------------------

Metodo posiciona o widget no container:
.pack(ipadx=#,ipady=#,padx=#,pady=#,side='#',fill=#,expand=#)

--------------------------------------------

Metodo que permite digitar e armazenar um texto de linha simples:
Entry(app)

--------------------------------------------

Metodo que permite digitar texto de varias linhas:
Text(app)
